## 1. 서비스 개요

### 리워들(Rewardle)이란?
**게임형 로컬 마케팅 플랫폼**

- 🎮 **게임**: 워들 게임 + 사과 게임으로 즐거운 경험
- 🏪 **로컬 비즈니스**: 동네 가게를 재미있게 홍보
- 💰 **리워드**: 게임으로 포인트 적립, 실제 혜택 제공
- 📱 **웹 기반**: 앱 설치 없이 바로 접근 가능

**핵심 가치 제안**
> "게임하고, 동네 가게 알아가고, 포인트 받고!"

## 2. 시장 분석 & 문제점

### 현재 시장 상황

**자영업자의 고민**
- 높은 온라인 광고 비용 (월 수십만원~수백만원)
- 낮은 광고 효과 (클릭률 1% 미만)
- 고객 유입 어려움
- 단골 고객 확보 어려움

**소비자의 불편함**
- 동네 맛집/좋은 가게 발견 어려움
- 광고는 많지만 신뢰도 낮음
- 리워드 앱은 많지만 재미없음
- 대형 프랜차이즈만 혜택 집중

**시장 기회**
- 🇰🇷 전국 자영업자 수: 약 550만 명
- 📈 로컬 마케팅 시장 규모: 연 10조원 이상
- 🎯 Z세대/밀레니얼: 게임형 콘텐츠 선호
- 💡 코로나 이후 로컬 비즈니스 재발견 트렌드

## 3. 솔루션: 리워들

### 어떻게 해결하나?

**자영업자에게**
✅ 저렴한 광고 비용 (기존 대비 1/10 수준)
✅ 게임으로 자연스러운 브랜드 노출
✅ 직접 퀴즈/게임 등록으로 능동적 참여
✅ 실제 방문 유도 (위치 정보 + 퀴즈)

**소비자에게**
✅ 재미있는 게임 콘텐츠
✅ 새로운 동네 가게 발견
✅ 실질적인 포인트 리워드
✅ 일일 미션으로 지속적 참여 유도

**Win-Win 구조**
```
자영업자: 효과적 홍보 + 신규 고객 유입
    ↓↑
리워들: 플랫폼 수수료 + 광고 수익
    ↓↑
소비자: 재미 + 포인트 + 새로운 발견
```

## 4. 핵심 기능

### 🎮 게임 시스템

**1. 워들 게임 (Wordle)**
- 가게 이름 맞추기 퍼즐
- 6번의 기회
- 초성 힌트 제공 (4번째 시도부터)
- 성공 시 5P + 추가 미션

**2. 사과 게임 (Apple Game)**
- 숫자 합 10 만들기
- 120초 제한 시간
- 글자 모으기 미션
- 확대/축소 기능 제공
- 성공 시 5P + 추가 미션

**3. 추가 미션**
- 가게 관련 퀴즈 (위치, 메뉴 등)
- 정답 시 추가 5P
- 자연스러운 가게 정보 학습

### 📅 일일 미션

**출석 체크**
- 게임 3회 플레이 후 출석 가능
- 기본 2P 지급
- 연속 출석 보너스:
  - 3일 연속: +1P
  - 7일 연속: +3P
  - 10일 연속: +5P
  - 30일 연속: +20P

**일일 게임 제한**
- 하루 최대 10회 플레이
- 과도한 포인트 획득 방지
- 건전한 게임 문화 유도

### 💰 포인트 시스템

**포인트 적립**
- 워들 게임 성공: 5P
- 사과 게임 성공: 5P
- 추가 미션 성공: 5P (각 게임)
- 일일 출석: 2P + 보너스

**포인트 사용**
- 제휴 가게 할인/쿠폰
- 기프티콘 구매
- 현금 환급 (향후 예정)

### 🏪 자영업자 기능

**퀴즈 등록 대시보드**
- 간단한 등록 인터페이스
- 가게명, 위치, 힌트 이미지 (선택)
- 게임별 정답 설정
- 추가 미션 퀴즈 작성

**실시간 통계**
- 게임 플레이 수
- 도달률
- 방문 전환율 (향후)

## 5. 게임 시스템 상세

### 워들 게임 메커니즘

**진행 방식**
1. 랜덤으로 가게 선택
2. 가게 이름 글자 수만큼 타일 생성
3. 키보드에서 글자 선택 (중복 제거)
4. 6번의 기회 동안 추리
5. 4번째 시도부터 초성 힌트 활성화

**피드백 시스템**
- 🟩 초록색: 정확한 위치의 정확한 글자
- 🟨 노란색: 글자는 맞지만 위치가 틀림
- ⬜ 회색: 정답에 없는 글자

**난이도 조절**
- 글자 수에 따른 자동 조절
- 초성 힌트로 적절한 난이도 유지

### 사과 게임 메커니즘

**게임 보드**
- 12x8 그리드 (96개 셀)
- 각 셀에 1~9 숫자 배치
- 숫자 분포 최적화:
  - 5: 24개 (가장 많음)
  - 4,6: 각 16개
  - 3,7: 각 10개
  - 2,8: 각 6개
  - 1,9: 각 2개 (가장 적음)

**플레이 방식**
1. 마우스/터치로 숫자 선택
2. 합이 10이 되는 2개 숫자 찾기
3. 사과 깨지면서 점수 획득
4. 숨겨진 글자 수집

**글자 수집 시스템**
- 사과 안에 가게명 글자 숨김
- 랜덤 위치에 배치
- 4점마다 30% 확률로 자동 획득
- 20점 이상 시 60% 확률로 증가

**UX 개선 요소**
- 확대/축소 버튼 (0.8x ~ 1.2x)
- 합이 10 아닐 때 부드러운 피드백
- 20초 무동작 시 힌트 표시

## 6. 포인트 & 리워드 체계

### 포인트 경제 설계

**일일 최대 획득 포인트**
- 게임 10회 × 10P (기본+추가) = 100P
- 출석 체크 = 2P + 보너스
- 일일 최대: 약 102~122P

**월간 예상 포인트**
- 매일 플레이 시: 약 3,000~3,500P
- 30일 연속 보너스 포함

**포인트 가치**
- 1P = 약 10원 상당
- 월 3,000P = 30,000원 상당
- 실제 사용 가치는 제휴처에 따라 변동

### 리워드 활용

**현재 제공**
- 게임 내 포인트 적립
- 통계 및 순위 (개발 예정)

**향후 제공 예정**
- 제휴 가게 할인 쿠폰
- 기프티콘 교환
- 현금 환급
- 가게 방문 인증 시 추가 보너스

## 7. 비즈니스 모델

### 수익 구조

**1. 광고 수수료 (주 수익원)**
- 자영업자 퀴즈 등록 수수료
- 월 구독형: 2만원/월 (무제한 등록)
- 건당 결제형: 5천원/건

**2. 프리미엄 노출**
- 게임 우선 노출권
- 추천 배너 광고
- 특정 지역 타겟팅

**3. 포인트 환급 수수료**
- 포인트 현금 전환 시 10% 수수료
- 제휴 가게 결제 시 수수료

**4. 데이터 분석 서비스**
- 지역별 소비 트렌드 리포트
- 업종별 성과 분석
- B2B 컨설팅

### 손익 분기 시뮬레이션

**가정**
- 월 구독 자영업자: 100명 × 2만원 = 200만원
- 건당 등록: 50건 × 5천원 = 25만원
- 프리미엄 광고: 20건 × 10만원 = 200만원
- **월 매출: 약 425만원**

**비용**
- 서버/인프라: 월 50만원 (Vercel + Supabase)
- 마케팅: 월 100만원
- 운영/인건비: 월 200만원
- **월 비용: 약 350만원**

**손익: +75만원**
→ 자영업자 100명 확보 시 손익분기점

## 8. 기술 스택

### Frontend
- **React 18 + TypeScript**: 타입 안정성과 최신 React 기능
- **Vite**: 빠른 개발 환경
- **Tailwind CSS**: 빠른 UI 개발
- **Framer Motion**: 부드러운 애니메이션

### Backend
- **Supabase**: 
  - PostgreSQL 데이터베이스
  - 실시간 API
  - Authentication (Google OAuth)
  - Row Level Security (RLS)

### 배포 & 인프라
- **Vercel**: 자동 배포, 글로벌 CDN
- **GitHub**: 버전 관리

### 주요 기술 특징
- **PWA 지원**: 웹앱으로 설치 가능
- **반응형 디자인**: 모든 디바이스 지원
- **오프라인 지원**: localStorage 활용
- **게스트 모드**: 로그인 없이도 플레이 가능

### 데이터 구조
```
📊 Database Schema
├── brands (브랜드/가게 정보)
├── user_points (사용자 포인트)
├── point_history (포인트 내역)
├── attendance (출석 체크)
└── game_plays (게임 플레이 기록)
```

## 💡 경쟁 우위

### vs 기존 리워드 앱
| 항목 | 리워들 | 기존 리워드 앱 |
|------|--------|---------------|
| 재미 | ⭐⭐⭐⭐⭐ 게임형 | ⭐⭐ 단순 적립 |
| 로컬 | ⭐⭐⭐⭐⭐ 동네 가게 | ⭐⭐ 대형 프랜차이즈 |
| 진입장벽 | ⭐⭐⭐⭐⭐ 웹 접근 | ⭐⭐⭐ 앱 설치 필요 |
| 광고효과 | ⭐⭐⭐⭐⭐ 자연스러운 노출 | ⭐⭐ 배너 광고 |

### 핵심 차별점
1. **게임화(Gamification)**: 단순 적립이 아닌 재미있는 게임
2. **로컬 중심**: 대형 브랜드가 아닌 동네 가게 지원
3. **낮은 진입장벽**: 웹 기반, 게스트 모드 지원
4. **자영업자 친화**: 직접 등록, 저렴한 비용
